# modelo de prediccion de abandono de clientes

# Objetivo 
"Identificar clientes en riesgo de abandonar el servicio con base en sus datos de comportamiento"

## Herramientas
 -Python
 -Pandas
 -Scikit-learn
 -Seaborn

## Modelo
 -Regresion Logistica

## Resultados
- Exactitud (Accuracy): 86%
- Recall (clase churn): 89%

## Insight clave
-los clientes con pagos atrasados y un alto numero de llamadas al soporte tienen mayor probabilidad de abandonar el servicio
